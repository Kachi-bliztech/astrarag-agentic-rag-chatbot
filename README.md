<h1>AstraRAG-Agentic RAG Chatbot(End-to-End</h1>



<h2>Description</h2>
This is an Agentic RAC chatbot, built as a complete end to end system and deployed on the cloud. This is a scalable cloude deployed AI agent capable of running complex task

<h2>Features</h2>

- </b>User-Friendly interface</b> 
- </b>AI Agents calling the RAG query tool</b> 
- </b>AI Agents displays sources of retrived answers</b> 
- </b>Scalability to other uses cases</b>




<h2>Tech Stack </h2>

- </b>CrewAI - Agents: crew AI will power our agents, enabling context aware and multi-step reasoning</b>
- </b>FastAPI - FastAPI-Backend: Fast API will serve as the back end, handling requests and orchestrates the workflow.</b>
- </b>Streamlit - for user Interface: Streamlit will be the front end user interface, giving users an interactive way to chat with the system from their browser.</b>
- </b>Groq - LLM: Grok will be the LLM provider, so that it gives us fast and efficient inference</b>
- </b>ChromaDB - Vectorstore: Chroma DB will act as the vector store, managing embeddings and retrieval.</b>
- </b>LlamaIndex - RAG Pipleline: Llama index will provide the RAC pipeline connecting the chatbot with external knowledge./b>
- </b>Docker - Container: To containerize this application, and it would be deployed in a more streamlined way./b>
- </b>AWS EC2 - Application Host/Deployment: This is used to host the application in the cloud, so it can be shared with others/b>
